# The New Sundial Demo Website

A modern, mobile-responsive demo website for The New Sundial, South Shields.

## Project structure

```text
new-sundial-demo/
├── index.html
├── styles.css
├── script.js
└── README.md
```

## Features

- Responsive restaurant and bar landing page
- Sticky navigation with mobile menu
- Smooth scroll and reveal animations
- Clickable cards that change state on click
- Hover colour interactions on cards
- Dojo booking link integration
- Google Maps embed at the end
- Review, food, event and contact sections

## How to run

Open `index.html` directly in a browser or deploy the folder to Netlify, Vercel, GitHub Pages or any static host.

## Notes

This is a concept/demo build for pitching purposes. Replace stock images with approved venue-owned images before final client delivery.
